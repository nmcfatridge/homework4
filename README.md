# homework4 uses Sass in TechFlow project

## H2

This home work was about learning Sass conversion in Visual Studio Code and incorporating it into our TechFlow web page. This involved a lot of debugging for me personally on PC, but eventually I got it to work after a lot of re watching videos and going through Microsofts' Command Prompt tutorials.

Going off of the intial Techflow homework 3 and introducing Sass:
1 - Convert the CSS project to Sass
2 - Organize styles into multiple Sass files.
3 - Use Sass variables throught the project.
4 - Use Sass nesting where appropriate.
5 - Compile Sass into a single css file.
6 - Maintain the existing design and responsive behavior of my website.

Project includes the following Sass Files
\_variables.scss for colors, fonts, widths, and breakpoints.
\_base.scss for reset and global styles.
\_header.scss for the site header and navigation.
\_hero.scss for the main hero section.
\_features.scss for feature/service sections.
\_clients.scss for the clients page.
\_footer.scss for footer styling.
\_responsive.scss for responsive layout adjustments.
styles.scss imports all partials and compiles into css/styles.css.
